# Food Delivery Platform - Clean Project Structure

## Essential Files (Keep These)
- `PROJECT_SUMMARY.md` - Project overview and architecture
- `SETUP_GUIDE.md` - Setup instructions for developers
- `RUN_INSTRUCTIONS.md` - How to run the application
- `init-db.sql` - Database initialization script
- `docker-compose.yml` - Docker setup for all services

## Service Directories (Keep These)
- `admin-frontend/` - Admin panel React app
- `frontend/` - User frontend React app
- `user-service/` - User authentication microservice
- `restaurant-service/` - Restaurant management microservice
- `order-service/` - Order processing microservice
- `payment-service/` - Payment handling microservice
- `delivery-service/` - Delivery tracking microservice

## Files to Remove
All `.bat`, `.html`, temporary `.sql` files, and debug documentation files are unnecessary for running the application and should be removed.
